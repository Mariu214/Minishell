# Description
The objective of the so_long project is to create a basic 2D video game. This project covers the fundamentals of the MiniLibX library as well as the basics of visual management. The player takes on the role of a skeleton searching for bones to reassemble their lover, all while dodging dogs eager to snatch those sweet and succulent bones.

# Instructions
To run this project, you must first provide a valid map as an argument. If the map is valid, a window containing the game will appear. You can use the arrow keys to move up, down, left, and right. You can exit the game by pressing the Escape key, clicking the close button (red cross) in the corner, or by completing the game. Touching a dog will result in a loss, while reaching the exit after gathering all collectibles will result in a victory.
